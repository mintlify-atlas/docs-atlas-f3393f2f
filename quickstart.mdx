</Warning>

This guide walks you through installing Deleo, authenticating with your Discord token, and deleting your first batch of messages.

<Steps>
  <Step title="Install Deleo">
    Clone the repository and install dependencies:

    ```bash
    git clone https://github.com/kars1996/deleo.git
    cd deleo
    bun install
    ```

    See the [Installation guide](/installation) for full prerequisites and platform notes.
  </Step>

  <Step title="Get your Discord token">
    Deleo requires your Discord user token to authenticate. The recommended way is to use the included token helper script:

    ```bash
    cd scripts
    pip install -r requirements.txt
    python token-helper.py --save
    ```

    This extracts your token from Discord or your browser cache and saves it to `~/.deleo_cached_token`. Deleo will load it automatically on the next launch.

    See the [token helper guide](/token/token-helper) for more details and troubleshooting.
  </Step>

  <Step title="Launch Deleo">
    Start the application:

    ```bash
    bun run dev
    ```

    The Deleo window opens and shows the **Authentication** screen.
  </Step>

  <Step title="Authenticate">
    On the authentication screen, choose one of two options:

    - **Auto-detect** — loads your token from `~/.deleo_cached_token` if you used the token helper
    - **Enter manually** — paste your Discord token directly, with an option to save it for future sessions

    After entering your token, Deleo validates it and scans your channels. See [Authentication](/usage/authentication) for more detail.
  </Step>

  <Step title="Select channels">
    Once authenticated, Deleo loads all channels where you have messages — DMs, group chats, and servers.

    Toggle the channels you want to clear, then configure your deletion settings:

    - **Deletion delay** — 500ms to 3 seconds between each deletion (higher values reduce rate-limit risk)

    See [Channel selection](/usage/channel-selection) for details on filtering and selecting channels.
  </Step>

  <Step title="Delete messages">
    Click **Delete selected** to begin. Deleo shows a live feed of deleted messages, failures, and overall progress.

    You can click **Stop** at any time to halt the process. See [Deleting messages](/usage/deleting-messages) for more information on the deletion process and [Stopping](/usage/stopping) for stopping behavior.
  </Step>
</Steps>
