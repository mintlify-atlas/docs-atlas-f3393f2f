## What is Deleo?

Deleo is a desktop GUI application built with [Electrobun](https://blackboard.sh/electrobun) that lets you delete your Discord messages in bulk. It provides a native desktop experience with real-time progress tracking, adjustable deletion speed, and support for DMs, group chats, and servers.

## How it works

Deleo uses a two-process architecture powered by Electrobun:

- **Bun backend** — handles Discord API calls, token management, message searching, and deletion logic
- **React frontend** — provides the desktop UI with real-time event updates from the backend via RPC

The backend communicates with Discord's API v9 to search for and delete your messages, respecting rate limits by introducing configurable delays between each deletion.
